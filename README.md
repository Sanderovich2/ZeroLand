# ZeroLand Bot

**ZeroLand** — это Telegram-бот, который переводит обычный русский язык на уникальный, эстетичный и «красивый» язык ZeroLand. Бот поддерживает два режима работы и работает как в обычных чатах, так и в режиме **Inline** (прямо в строке ввода).

### Особенности
* 🔄 **Два режима:** Обычный (простая замена букв) и Декоративный (добавляет обрамление в виде символов `⟦ ⟧`).
* ⚡ **Inline-режим:** Работает прямо в любом чате Telegram. Напиши `@zeroland_translatebot твой текст`, чтобы увидеть варианты перевода.
* 🛠 **Панель администратора:** Управляй ботом, делай перерыв, отправляй массовые рассылки или обновляй код прямо через Telegram.
* 🚀 **Защита от спама:** Встроенный ratelimit (защита от слишком частых запросов).

## Требования (Зависимости)

Перед запуском убедись, что у тебя установлен **Python 3.7+** и библиотека `pyTelegramBotAPI`.

*(Остальные библиотеки: json, time, threading — являются встроенными в Python, устанавливать их не нужно).*

## Как запустить (Полный тутор)

### Шаг 1. Создай бота у @BotFather
1. Зайди в Telegram к боту [@BotFather](https://t.me/BotFather).
2. Введи `/newbot` и следуй инструкциям.
3. После создания BotFather выдаст тебе **Токен**.

### Шаг 2. Создай файлы конфигурации
Бот использует файлы для сохранения настроек. Создай в той же папке, где лежит `bot.py`, два пустых файла:
1. `user_modes.json`
2. `bot_status.json`

> **Важно:** Если бот не может найти эти файлы, он может выдать ошибку при запуске.

### Шаг 3. Запуск бота
Открой командную строку (Терминал / CMD / PowerShell) в папке с проектом и запусти файл `bot.py`.

**Готово!** Бот запущен. Перейди в Telegram и напиши своему боту команду `/start`.

## 🕹 Команды бота

**Для обычных пользователей:**
* `/start` — Открыть главное меню с выбором режима и алфавитом.
* `/normal` — Включить обычный режим перевода.
* `/decorative` — Включить декоративный режим (с обрамлением).
* `/alphabet` — Показать алфавит ZeroLand.
* `/myid` — Показать ваш ID.
* `/help` — Справка по боту.

**Для Администратора** (только для ID, указанного в `ADMIN_ID`):
* `/admin_panel` — Открыть панель управления:
  * `⏸ Перерыв` — Делает бота недоступным для всех.
  * `▶️ Нет перерыва` — Снова включает бота.
  * `📤 Отправить всем` — Останавливает бота, чтобы сделать массовую рассылку, после чего автоматически включает обратно.
  * `🔄 Обновить бота` — Позволяет заменить код `bot.py` прямо через Телеграм (бот перезагрузится с новым кодом).

## ⚙️ Обновление и устранение неполадок

* **Ошибка "AttributeError" или "ImportError"**: Убедись, что ты установил библиотеку `pyTelegramBotAPI`, а не просто `telebot`.
* **Ошибка с JSON файлами**: Убедись, что файлы `user_modes.json` и `bot_status.json` существуют в папке с ботом и в них написано `{}` (пустые фигурные скобки), либо бот создаст их сам при первом запуске.
* **Защита от спама**: В коде установлена задержка 0.35 секунды между отправкой сообщений. Если бот "тормозит", можно уменьшить это значение в строке `MIN_REQUEST_INTERVAL = 0.35`.
